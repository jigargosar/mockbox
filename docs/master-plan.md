# Mockbox — Master Plan

Balsamiq-style wireframe mock builder. Sketchy, hand-drawn aesthetic. Fast, keyboard-driven, runs entirely in-browser.

1. Infinite canvas with pan (middle-click / space+drag) and zoom (scroll wheel / pinch)
2. Hand-drawn sketchy line rendering (roughjs or similar)
3. Wireframe color palette (grayscale + blue accent for links)
4. Handwriting font for text
5. Sidebar component palette organized by category
6. Drag from palette onto canvas to place
7. Snap-to-grid with configurable grid size
8. Click to select, shift+click for multi-select, drag to marquee select
9. Move selected via drag or arrow keys (shift+arrow for larger steps)
10. Resize via handles with aspect ratio lock (shift)
11. Duplicate (Ctrl+D), delete (Del/Backspace)
12. Context-sensitive property panel for selected component
13. Dimensions — x, y, width, height (numeric input)
14. Typography — font size, weight, alignment
15. Content editing — text, placeholder, options (for dropdowns/radios)
16. Style overrides — fill, border, corner radius, opacity
17. Unlimited undo/redo (Ctrl+Z / Ctrl+Shift+Z) with command-based history stack
18. Copy/paste (Ctrl+C/V)
19. History panel showing named actions
20. Layers panel showing z-order of all components on current page
21. Drag to reorder layers, toggle visibility, toggle lock
22. Group/ungroup components (Ctrl+G / Ctrl+Shift+G)
23. Name layers for clarity
24. Lock/unlock components to prevent accidental edits
25. Smart alignment guides (snap to edges/centers of nearby components)
26. Rulers along top and left edges
27. Minimap for orientation on large canvases
28. Page list sidebar — add, rename, reorder, duplicate, delete pages
29. Copy/paste across pages
30. Hotspot links — draw link regions that navigate to other pages on click
31. Presentation mode — click through linked pages as interactive prototype
32. Auto-save to IndexedDB (local-first, no server required)
33. Manual export/import as `.mockbox` JSON file
34. Recent projects list on home screen
35. Export current page or all pages as PNG
36. Export as PDF (multi-page document)
37. Export as SVG
38. Copy selection as image to clipboard
39. Component state — default, hover, disabled, active, focused
40. Search/filter within palette
41. Favorites for frequently used components
42. Toggle between sketchy and clean rendering modes
43. Comprehensive keyboard shortcut system
44. Shortcut cheatsheet overlay
45. All canvas operations accessible via keyboard
46. Screen reader announcements for selection changes
47. High contrast mode option
